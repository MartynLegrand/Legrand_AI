# Quick Start Guide

Get Legrand AI up and running in 5 minutes!

## Step 1: Get the Code

```bash
git clone https://github.com/MartynLegrand/Legrand_AI.git
cd Legrand_AI
```

## Step 2: Start a Local Server

Choose one of these methods:

### Python (Easiest)
```bash
python -m http.server 8000
```

### Node.js
```bash
npx http-server
```

### Or just double-click `index.html` in your browser!

## Step 3: Open in Browser

Navigate to: **http://localhost:8000**

## Step 4: Get an OpenAI API Key

1. Go to https://platform.openai.com/api-keys
2. Sign up or log in
3. Create a new API key
4. Copy the key (starts with `sk-...`)

## Step 5: Configure Legrand AI

1. Click the ⚙️ settings icon in the bottom-right
2. Paste your API key
3. That's it! Start chatting!

## First Conversation

Try these example prompts:

- "Hello! What can you do?"
- "Tell me a joke"
- "What is 3D AI technology?"
- "Help me understand quantum computing"

## Voice Features (Optional)

1. Click the 🎤 microphone icon
2. Allow microphone access when prompted
3. Speak your message
4. It will automatically transcribe and send!

## Tips

- ✨ Watch the 3D avatar animations
- 💬 Clear chat with the 🗑️ icon
- 🔊 Enable voice responses in settings
- 📱 Works on mobile too!

## Troubleshooting

### "API request failed"
→ Check your API key and internet connection

### Voice not working
→ Use Chrome or Edge browser for best results

### Avatar not showing
→ Refresh the page and check browser console
